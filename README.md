# 3D 梯度下降視覺化：探索 AI 行為的機率地形

![ss](./screenshot.png)

這不僅是一個 3D 梯度下降 (Gradient Descent) 的視覺化工具，更是一個互動實驗，用以探索 AI 和大型語言模型 (LLM) 行為的核心機制：Noise (噪聲) 與 Entropy (熵)。

## 本專案旨在回答一個核心問題：為什麼 AI 看似具有「創造力」？

> 核心理念：AI 是「結構化機率地形 + 可控隨機流動」


根據 Andrej Karpathy 的觀點與我們此前的深入討論，AI (尤其是 LLM) 的「模式崩潰」(Model Collapse)——例如 ChatGPT 總講一樣的笑話——是因為其推理過程中缺乏足夠的 Entropy。

## 這個視覺化工具將這個抽象概念具象化

- 3D 地形 (Landscape): 代表 AI 模型的「機率分佈」或「損失表面」(Loss Surface)。訓練過程就是為了「雕刻」出這個地形。

- 紅球 (Optimizer): 代表 AI 在一次「推理」(Inference) 或「生成」過程中，試圖找到最佳答案（地形最低點）的路徑。

- 凹坑 (Basins): 代表「局部最小值」(Local Minima) 或「慣性思維」。這就是 AI 陷入 Model Collapse 的地方——一個「還不錯」但「非最佳」的答案。

# 如何使用這個實驗？

## 1. 兩種地形 (Functions)

- 簡單地形 (Simple Bowl): 這是一個「凸函數」(Convex)，它只有一個「全域最小值」(Global Minimum)。在這種地形上，任何演算法都能輕易找到答案。這代表一個「簡單、已收斂」的任務。

- 複雜地形 (Complex Landscape): T這是本實驗的核心。它充滿了「局部最小值」（小凹坑）。這完美地模擬了 LLM 擁有的複雜知識結構。

## 2. 關鍵參數：熵的注入

- 請切換到「複雜地形」並進行以下實驗：

    ### 實驗 A：缺乏 Entropy (AI 的模式崩潰)

    將 Noise / Entropy 設為 0。

    將 Momentum (動量) 設為一個較低的值 (例如 0.1)。

    反覆點擊 重置模擬 (Reset)。

    觀察： 球會從不同的隨機位置開始，但它幾乎總是滾入離它最近的那個凹坑並停在那裡。它無法靠自身力量越過山丘去尋找那個真正最低的「全域最小值」。

    這就是 Model Collapse。 這等同於您要求 Temperature = 0 的 AI，它會給您一個可預測、確定性、但缺乏新意的答案。

    ### 實驗 B：注入 Entropy (AI 的「創造力」)

    保持在「複雜地形」。

    將 Noise / Entropy 調高 (例如 2.0 或更高)。

    反覆點擊 重置模擬 (Reset)。

    觀察： 球開始劇烈地隨機「抖動」。這個「噪聲」能量（來自 Noise 參數的隨機向量）會不斷地「踢」這顆球。這股能量足以讓它跳出局部凹坑，越過山丘，去探索地形的其他區域。

    這就是 Entropy 的力量。 這等同於您使用 Temperature > 0 與 AI 互動。您注入的「隨機性」迫使模型離開其最安逸的（高機率）路徑，去探索那些它「知道」但「不常說」的知識（即「知識的長尾」）。

### 3. 其他參數

- Learning Rate (學習率): 球滾動的「步伐」大小。太高會導致「梯度爆炸」，球會飛出地圖。

- Momentum (動量): 球的「慣性」。高動量有助於「衝過」小山丘，是另一種形式的探索（但 Noise 是更根本的隨機性來源）。

- 實時數據 (Data Log): 觀察 梯度 (Grad) [gx, gz] 數值。這就是地形對球施加的「拉力」。Noise 的作用就是在這個「拉力」向量上，額外添加了一個「隨機」向量。

# 結論：為什麼 AI「像人」？

我們在討論中曾提到，Temperature = 0 的 AI 輸出讓人感覺「僵硬」且「不像人」。

這個模擬揭示了底層 logique：
```
一個「像人」的回答，並非因為 AI 有意識，而是因為它的輸出路徑（球的軌跡）具有「有規則的隨G機性」。這種隨機性來自 Entropy 的注入，使其軌跡（生成的文本）免於陷入可預測的、確定性的「凹坑」。

這個視覺化工具證明了：我們所感知的 AI「創造力」，本質上是一種「受控的隨機性」 (Controlled Stochasticity)，它在一個被訓練好的「結構化機率地形」上進行探索。
```
